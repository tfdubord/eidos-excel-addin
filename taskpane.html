Tag List Expansion

  1. Create the template sheets.

  2. Populate the TagsIn sheet from PLC tag export.
  3. Use the Get Unique DTs button to generate a list of data-types from your TagsIn sheet.

  4. Defining which members (.members) you want to include for each data type by filling in in Column B. Separate out multiple members with semicolons (.PV;.SP).
  5. Then generate the expanded tag list with this button.
